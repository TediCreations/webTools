# Development tools

> Base docker image for all my web development tools.

## Usage

Available from:
- [Dockerhub](https://hub.docker.com/r/tedicreations/webtools)
- [Github](https://github.com/TediCreations/webTools)

```sh
docker pull tedicreations/webtools
```

## Development

### Build

```sh
make
```

### Run

```sh
make run
```
